## 🚀 快速部署

### 第一步：注册腾讯云账号
1. 访问 https://cloud.tencent.com
2. 注册账号（免费注册）
3. 完成实名认证

### 第二步：开通对象存储 COS 服务
1. 登录后搜索"对象存储 COS"
2. 点击"立即开通"
3. 免费额度：50GB 存储 + 10GB/月 流量

### 第三步：创建存储桶
1. 点击"创建存储桶"
2. 填写信息：
   - **名称**: `xuekao`（或其他您喜欢的名字）
   - **地域**: 选择离您近的（如：广州/北京/上海
   - **访问权限**: 公有读私有写
3. 点击"创建"

### 第四步：上传网站文件
1. 进入刚创建的存储桶
2. 点击左侧"文件管理"
3. 点击"上传文件"
4. 找到 `d:\me\dist` 文件夹
5. **上传 dist 文件夹里的所有文件（包括 index.html, bundle.js 等）
6. 等待上传完成

### 第五步：开启静态网站托管
1. 左侧点击"基础配置" → "静态网站"
2. 点击"编辑"
3. 开启"静态网站"开关"
4. 配置：
   - **索引文档**: `index.html`
   - **错误文档**: `index.html`（重要！SPA 应用需要）
5. 点击"保存"

### 第六步：获取访问地址！
- 腾讯云会提供访问地址，格式类似：
  `https://xuekao-1234567890.cos.ap-guangzhou.myqcloud.com
- 或者在浏览器中打开即可访问！

## 🔄 更新网站
1. 修改代码
2. 运行 `npm run build`
3. 在 COS 文件管理中重新上传 dist 文件
4. 覆盖旧文件即可
