---

## 🚀 部署步骤

### 第一步：注册阿里云账号
1. 访问 https://www.aliyun.com
2. 注册账号（免费注册）
3. 完成实名认证

### 第二步：开通对象存储 OSS 服务
1. 登录后搜索"对象存储 OSS"
2. 点击"立即开通"
3. 有免费额度可以使用

### 第三步：创建存储桶
1. 点击"创建 Bucket"
2. 填写信息：
   - **Bucket 名称**: `xuekao`（或其他名字）
   - **地域**: 选择离您近的（推荐：华南1（广州）/华东1（杭州）/华北2（北京））
   - **存储类型**: 标准存储
   - **读写权限**: 公共读
3. 点击"确定"

### 第四步：上传网站文件
1. 进入刚创建的 Bucket
2. 点击左侧"文件管理"
3. 点击"上传文件"
4. 找到 `d:\me\dist` 文件夹
5. **上传 dist 文件夹里的所有文件（包括 index.html, bundle.js 等）**
6. 等待上传完成

### 第五步：开启静态网站托管
1. 左侧点击"数据管理" → "静态页面"
2. 点击"设置"
3. 开启"静态页面托管"
4. 配置：
   - **默认首页**: `index.html`
   - **默认 404 页**: `index.html`（重要！SPA 应用需要）
5. 点击"确定"

### 第六步：获取访问地址！
- 在静态页面设置页面可以看到访问地址
- 格式类似：`https://xuekao.oss-cn-guangzhou.aliyuncs.com`
- 直接在浏览器中打开即可访问！

## 🌐 关于自定义域名

### ❌ 国内节点 + 自定义域名
- 必须备案，不管是国内域名还是国外域名
- 备案需要 3-20 个工作日

### ✅ 香港节点 + 自定义域名（无需备案）
1. 创建 Bucket 时选择 **香港** 地域
2. 绑定自定义域名
3. 无需备案

## 🔄 更新网站
1. 修改代码
2. 运行 `npm run build`
3. 在 OSS 文件管理中重新上传 dist 文件
4. 覆盖旧文件即可
