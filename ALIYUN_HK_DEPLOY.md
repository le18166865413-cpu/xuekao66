## 🚀 详细部署步骤

### 第一步：注册阿里云账号
1. 访问 https://www.aliyun.com
2. 点击右上角"注册"
3. 使用手机号或邮箱注册
4. 完成实名认证

### 第二步：开通对象存储 OSS 服务
1. 登录后在顶部搜索栏搜索"对象存储 OSS"
2. 点击进入 OSS 产品页
3. 点击"立即开通"（免费额度）

### 第三步：创建香港地域的存储桶
1. 在 OSS 控制台点击"创建 Bucket"
2. **重要！地域选择：**
   - 选择：**中国（香港）**
3. Bucket 名称：`xuekao`（或其他您喜欢的名字）
4. 存储类型：**标准存储**
5. 读写权限：**公共读**
6. 点击"确定"

### 第四步：上传网站文件
1. 进入刚创建的 Bucket（香港地域的）
2. 点击左侧菜单"文件管理"
3. 点击"上传文件"
4. 找到您的项目文件夹：`d:\me\dist`
5. **上传 dist 文件夹里的所有文件**（包括 index.html, bundle.js, CSS 等）
6. 等待上传完成

### 第五步：开启静态网站托管
1. 左侧点击"数据管理" → "静态页面"
2. 点击"设置"
3. 开启"静态页面托管"开关
4. 配置：
   - **默认首页**: `index.html`
   - **默认 404 页**: `index.html`（重要！SPA 应用需要这个配置）
5. 点击"确定"

### 第六步：获取访问地址（测试）
- 在静态页面设置页面可以看到阿里云提供的默认访问地址
- 格式类似：`https://xuekao.oss-cn-hongkong.aliyuncs.com`
- 在浏览器中打开测试一下，确保能正常访问

## 🌐 绑定国外自定义域名（可选）

### 准备工作
需要您已经拥有一个国外域名（如 .com, .net, .org 等）

### 步骤：
1. 在 Bucket 页面左侧点击"域名管理"
2. 点击"绑定域名"
3. 输入您的国外域名（如：`xuekao.com`）
4. 点击"确定"
5. 阿里云会提示您配置 CNAME 解析

### 配置 DNS（在您的域名注册商处）
1. 登录您的域名注册商网站
2. 找到 DNS 管理/域名解析
3. 添加 CNAME 记录：
   - 主机记录：`@` 或 `www`
   - 记录类型：`CNAME`
   - 记录值：阿里云提供的 Bucket 域名（如：`xuekao.oss-cn-hongkong.aliyuncs.com`）

### 配置 HTTPS（可选）
1. 在域名管理页面点击"证书托管"
2. 上传 SSL 证书或申请免费证书
3. 开启 HTTPS 访问

## 🔄 更新网站

### 步骤：
1. 修改您的代码
2. 运行 `npm run build`
3. 在 OSS 文件管理中删除旧文件
4. 重新上传新的 dist 文件夹内容
5. 覆盖即可
